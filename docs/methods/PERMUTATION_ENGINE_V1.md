# PLS Structural-Score Permutation Specification v1

Status: validated for the documented QuickPLS v0.9.0-rc.1 supported inference scope. Broader permutation designs outside this contract remain unsupported until independently audited.

## Scope

`freedman_lane_permutation_v1` provides a two-sided conditional permutation test for every direct path in a converged PLS-PM result. It tests structural coefficients using the fixed construct scores produced by the original measurement model. It is not a full measurement-model re-estimation test, MGA, MICOM, or a substitute for bootstrap confidence intervals.

## Freedman-Lane Procedure

For focal path `X_j -> Y`, let the remaining direct predecessors of `Y` be nuisance predictors `X_-j`.

1. Fit an intercept regression of the original target construct score `Y` on `X_-j` using column-pivoted QR. With no nuisance predictor, the fitted value is the target-score mean.
2. Store fitted values and residuals from that reduced equation.
3. For permutation index `p`, generate a Fisher-Yates permutation of the residual positions from a ChaCha20 stream derived from `(master_seed, path_parameter_identity, p)`.
4. Form `Y*_p = fitted_reduced + permuted_residual` and regress `Y*_p` on all original direct predecessors, including `X_j`.
5. The two-sided exceedance count is `count(|beta*_j| >= |beta_j|)`. The reported finite-sample probability is `(exceedances + 1) / (P + 1)`.

Every focal path has an independently domain-separated stream. Replicate and parameter ordering are stable across worker counts. A requested analysis uses 99 through 10,000 permutations; zero disables the method. Any QR or replicate failure fails the permutation artifact instead of silently changing its denominator. Cancellation discards partial output.

## Persistence

Projects store only the method version, plan, canonical path identity, original coefficient, exceedance count, corrected probability, and permutation count. The `pls_pm_v3` result envelope can contain bootstrap and permutation artifacts independently. V1 and v2 PLS envelopes remain readable.

## Interpretation Boundary

The null is that the focal structural-score coefficient is zero conditional on the nuisance construct scores and the already-estimated measurement solution. Measurement-score uncertainty is not regenerated by this procedure. QuickPLS labels the table "Freedman-Lane permutation" and keeps it experimental until its conditional type-I error and power are qualified on documented data-generating processes.
